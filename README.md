戦争をしているなかで「師団の補給が足りない！戦えない！」そんな悩みはありませんか？  
このMODは、その悩みを改善するために作られています。

## 📝 概要 / Overview
このMODは、独自のディシジョンを追加し、兵站システムを戦略的に強化します。ゲームバランスを保つため、強力なバフと引き換えに一定のデバフが含まれるように設計されています。

### 🔹 主な変更点 / Key Features
* **建築速度増加 (Increased Construction Speed):** インフラ整備を加速させます。
* **物資使用量減少 (Reduced Supply Consumption):** 少ない物資で師団を維持します。
* **補給切れ減少 (Reduced Supply Penalty):** 補給不足による悪影響を軽減します。
* **物資補給猶予増加 (Increased Out-of-Supply Grace Period):** 補給が切れてからデバフが乗るまでの時間を延ばします。

*※ゲームバランス調整のために、それぞれに一定のデバフを含みます。*

## 🛠 導入方法 / Installation
1. このリポジトリをクローンまたはSteamからダウンロードし、HoI4のModフォルダに配置してください。
   `Documents/Paradox Interactive/Hearts of Iron IV/mod/`
2. `.mod` ファイルのパスが正しくディレクトリを指しているか確認してください。
3. ParadoxランチャーでModを有効にします。

## 🌐 言語対応 / Language Support
日本語と英語に対応しています。  
*※英語に関しては機械翻訳を利用しているため、一部不自然な表現が含まれる可能性があります。*
